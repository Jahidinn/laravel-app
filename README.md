## Laravel web app ()

Laravel is a web application framework with expressive, elegant syntax. We believe development must be an enjoyable and creative experience to be truly fulfilling. In this Laravel web application there are several features in it, namely:

-   Login and Registration user
-   Front-end view blog posts.
-   Create article.
-   Edit, update and delete article.
-   Category management.
-   Back-end article view.
-   Email verification.
-   User Atirization / role access.
